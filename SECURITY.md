# Security

## Trust model

Blue Pearmain is a personal tool designed for single-user, trusted local network use. The reviewer UI is not hardened for internet-facing or multi-user deployment — see the Scope and trust model section in the README.

## Reporting a vulnerability

There is no formal bug-bounty programme. If you find a security issue, please report it privately rather than opening a public GitHub issue:

- Use [GitHub's private vulnerability reporting](https://github.com/cdevers/Blue-Pearmain/security/advisories/new) for this repository, or
- Email the maintainer directly at cdevers@pobox.com.

Please include a description of the issue, steps to reproduce, and any relevant environment details. You can expect an acknowledgement within a few days.
